# 👑 Queen's Cup — Como publicar no Vercel

## Passo a passo (5 minutos)

### 1. Crie uma conta grátis no GitHub
- Acesse: https://github.com
- Clique em "Sign up"
- Use seu email e crie uma senha

### 2. Crie um repositório no GitHub
- Após entrar, clique no "+" no canto superior direito
- Clique em "New repository"
- Nome: `queens-cup`
- Deixe como "Public"
- Clique em "Create repository"

### 3. Faça upload dos arquivos
- Na página do repositório, clique em "uploading an existing file"
- Arraste TODA a pasta `queenscup` para a área de upload
- Clique em "Commit changes"

### 4. Publique no Vercel
- Acesse: https://vercel.com
- Clique em "Sign up" → entre com sua conta do GitHub
- Clique em "Add New Project"
- Selecione o repositório `queens-cup`
- Clique em "Deploy"
- Aguarde ~1 minuto ☕

### 5. Seu link estará pronto!
- O Vercel vai gerar um link como: `queens-cup.vercel.app`
- Você pode personalizar o nome nas configurações do projeto

## Para atualizar o ranking no futuro
1. Abra o arquivo `src/App.jsx` no GitHub
2. Clique no ícone de lápis (editar)
3. Altere os dados do ranking
4. Clique em "Commit changes"
5. O Vercel republica automaticamente em ~30 segundos!
